Endocrinology Self-Assessment Program

Start Time: 12:00

End Time: 13:00

This event takes place is Room 2270 and is either an Endocrine Self- Assessment Program (ESAP) or an American College Self-Assessment Program (ASAP). The ESAP is a self-study program aimed at physicians seeking certification or recertification in endocrinology, program directors interested in a testing and training instrument, and individuals simply wanting a self-assessment and broad view of endocrinology. The ASAP is a self-study program that provides updated information on the evaluation, diagnosis, treatment and management of endocrine disease.
